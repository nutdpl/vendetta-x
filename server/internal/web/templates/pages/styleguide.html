{{/* ============================== BUTTONS ================================ */}}

When to use: .btn-primary for the one main action; .btn default outline; .btn-ghost tertiary; add .btn-sm for inline/compact (e.g. file download).

{{/* ============================== FORMS ================================== */}}

When to use: .field wraps a label+input. Inputs/textareas style automatically. .onelineform is the inline single-row form (the wall). .postform is the board reply box.
